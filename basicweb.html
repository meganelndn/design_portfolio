01.04.01 Responsive Website

FOCUS To build a website that has a responsive layout, meaning a layout that adapts to different sized viewports.

The design and content was inspired by a particular style, in this case it follows the design principles of punk aesthetics. The emphasis was put on creating a visually effective website that provides comfortable viewing and using experience on different devices.

Applying the “mobile first” principle in creation of the website. Starting with small-screen device format and moving on to larger viewport sizes.

GAINS More advanced use of CSS, in particular use of grids and media queries. The large-screen version of the website is built on three column layout, while the mobile-device version is built on two grid columns.

Understanding of design principles - colour schemes, importance of element layout and gestalt principles.

Use of XD prototyping and user testing.
