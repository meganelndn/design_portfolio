Basic UX

04.01.01/02 Design Sprint and
Web Prototype

FOCUS Creating a functional high fidelity prototype for a mobile version of a KEA Alumni platform for Fronter. Focusing on one main goal - imporving employment possibilities for KEA graduates.

Creating the prototype in five days while working in a team of four people and being led by a facilitator. Going through all sprint stages day by day - idea generation, brainstorming, sketching, testing and implementing.

Applying input forms to user interface both in the prototypes and in the HTML/CSS code.

GAINS Adapting to work in a fast paced rhythm in which ideas need to be generated quickly and the decisions need to be made on the spot.

Working better in a team, where roles are more clearly distributed and there is a certain person responsible for making the final decisions.

Understanding all steps of a design sprint and getting used to the prototype creation and user testing.
