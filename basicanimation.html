02.02.01 Storytelling and
Character Design

FOCUS To create a coherent visual animated story with a message.

The story had to follow the basic stages of introduction, build up, climax and fade out.

The visuals of the animation were based on particular style, in this case the style was "LIMBO".

An animation was created using sprite sheet drawn in Illustrator and HTML+CSS code.

GAINS Use of Adobe Illustrator. Learning to use different strokes, figures, forms etc.

Learning about basic principles of animation - dimensions, proportions, movements and others.Applying storyboard, style tile and other tools.

Drawing sprite sheets and using sprite animation to create movement/animation effect using HTML and CSS.
